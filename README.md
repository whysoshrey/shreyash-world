# shreyash-world

Vite + React + TypeScript + Framer Motion. Deploys to GitHub Pages via Actions.

## Local run
```bash
npm install
npm run dev
```

## Edit content
- `public/content.json`

## PDFs
Put these into `public/`:
- `cv.pdf`
- `lor-pritika.pdf`
- `lor-deepak.pdf`

## Deploy
GitHub repo:
- Settings → Pages → Source: GitHub Actions
Push to `main` → auto deploy.
